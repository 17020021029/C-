# 比较版本
## description
版本号是版本的标识号。可以假设版号只含数字和字符'.'，字符'.'代表的不是小数点，只是用来分隔每个数字。比如，2.5表示的是"the fifth second-level revision of the second first-level revision"，而不是 "two and a half" 或者 "half way to version three"。例如，按版本先后顺序排序，有0.1 < 1.1 < 1.2 < 1.10 < 13.37。

现在的任务是，比较两个版本的版本号，输出较新的版本号。
input</br>
输入为两行。分别为两个版本号，不含空格，</br>
output</br>
输出较新版本的版本号。
## 代码实现
错误代码:根本没法输出</br>
思路：比较两个字符串的大小，长的一定是新的；遍历，找到不同的，即输出新版本号
```
#include<iostream>
#include<string.h>
using namespace std;
int main()
{
	char a[100],b[100];
	int i=0,j=0;
	while(a){
		cin>>a[i];
		i++;}
	while(b){
		cin>>a[j];
		j++;
	}
	int len1,len2;
	len1=strlen(a);
	len2=strlen(b);
	if(len1>len2)
		for(int i=0;i<len1;i++){
		cout<<a[i];
		cout<<endl;}
	if(len1<len2)
		for(int i=0;i<len2;i++){
		cout<<b[i];
		cout<<endl;}
	else{
		for(int j=0;j<len1;j++)
		{
			if(a[i]==b[i])continue;
			if(a[i]>b[i])
			{
				for(int k=0;k<len1;k++){			
				cout<<a[k];
				cout<<endl;}
			}
			if(a[i]<b[i])
			for(int k=0;k<len2;k++){
				cout<<b[k];
				cout<<endl;
			}
		}
	}
	return 0;
 } 
```
